1. 列表项出现以下问题

```javascript
ERROR [backend internal] load metadata for docker.io/library/python:3.11-slim                   22.4s  ------  > [backend internal] load metadata for docker.io/library/python:3.11-slim: ------ Dockerfile.worker:2  --------------------     1 |     # =
```

docker经常出现这种拉去镜像源导致的网络问题，在这种时候我们需要再docker的setting中找到engine设置，然后在其中加入以下选项：

```javascript
{
  "builder": {
    "gc": {
      "defaultKeepStorage": "20GB",
      "enabled": true
    }
  },
  "experimental": false,
  "registry-mirrors": [
    "https://ccr.ccs.tencentyun.com",
    "https://docker.rainbond.cc",
    "https://elastic.m.daocloud.io",
    "https://elastic.m.daocloud.io",
    "https://docker.m.daocloud.io",
    "https://gcr.m.daocloud.io",
    "https://ghcr.m.daocloud.io",
    "https://k8s-gcr.m.daocloud.io",
    "https://k8s.m.daocloud.io",
    "https://mcr.m.daocloud.io",
    "https://nvcr.m.daocloud.io",
    "https://quay.m.daocloud.io"
  ]
}
```
最终保存运行即可
